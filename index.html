Sonic Surfer (2020)

Sonic Surfer is a multimedia game inspired by early iphone-games such as temple run and flappy bird. The game is controled through a pressure senstive pad that the player stands on. Sonic Surfer relies primarily on spatial audio information encoded for a pair of headphones. The user leans towards "good" sounds and avoids the "bad" ones.

SimpleDelay(2020)

A tape delay plug-in built in JUCE. Controls include delay time, wet level, feedback level, and soft clipping threshold with pre/post processing option. Implementation of a cirrcular buffer to manage delayed signal.
